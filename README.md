# D-VotingSystem
A blockchain application developed using Truffle Framework.

Built for Student Council Elections.


Requirements:
nodeJS,Truffle,Ganache,Metamask client

- Install dependencies using
   - npm install
   - truffle migrate --reset
   - truffle test
   - npm run dev
